# CodeAlpha_BasicChatbot

# 🤖 Basic Chatbot using Python

## 📌 Project Description
This is a simple rule-based chatbot built using Python.  
It responds to predefined user inputs like "hello", "how are you", and "bye".

---

## 🚀 Features
- Simple conversational chatbot
- Rule-based responses using if-elif statements
- Continuous chat loop until user exits
- Beginner-friendly Python project

---

## 🧠 Concepts Used
- Functions
- If-Else conditions
- Loops (while loop)
- User Input/Output

---

## 💻 How It Works
1. User enters a message
2. Program checks the input using rules
3. Bot returns a predefined response
4. Loop continues until user types "bye"

---

## ▶️ Example

You: hello  
Bot: Hi! How can I help you?

You: how are you  
Bot: I'm fine, thanks! What about you?

You: bye  
Bot: Goodbye! Have a nice day 😊
